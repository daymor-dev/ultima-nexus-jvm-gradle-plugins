= Plugin Reference
:toc: left
:toclevels: 3

Complete catalog of all 78 Ultima Nexus JVM Gradle plugin IDs, organized by category.

All plugin IDs share the prefix `dev.daymor.ultimanexus.jvm.gradle.`.
Settings plugins are applied in `settings.gradle.kts`; all others are applied in `build.gradle.kts`.

== Settings Plugins

Settings plugins are applied in `settings.gradle.kts` and configure the build environment before any project build scripts are evaluated.

NOTE: Version catalogs (`libs`) cannot be used in `settings.gradle.kts` plugins blocks.
Use the direct `id("...") version "..."` syntax.

== Bundle Plugins

Bundles compose multiple feature plugins into a single plugin for convenience.
They are organized in layers, where each level builds on the one below.
