= Ultima Nexus - Native Module
:toc: preamble
:toclevels: 2

Native image convention plugins for JVM projects.

== Overview

The `plugins-native` module provides GraalVM Native Image build conventions.
These plugins are framework-agnostic and can be used with Spring Boot, Micronaut,
Quarkus, or any JVM application that supports native compilation.

All plugins support configuration via extension DSL or gradle.properties.

== Plugins

[cols="1,3"]
|===
|Plugin ID |Description

|`dev.daymor.ultimanexus.jvm.gradle.feature.graalvm-native`
|GraalVM Native Image compilation and test execution
|===

== Installation

Add to your version catalog (`gradle/libs.versions.toml`):

[source,toml]
----
[versions]
ultima-nexus-plugins = "latest.release"

[plugins]
ultimanexus-jvm-feature-graalvm-native = { id = "dev.daymor.ultimanexus.jvm.gradle.feature.graalvm-native", version.ref = "ultima-nexus-plugins" }
----

== GraalVM Native Image Plugin

Applies the `org.graalvm.buildtools.native` plugin and configures default
native image build settings.

=== Usage

[source,kotlin]
----
plugins {
    alias(libs.plugins.ultimanexus.jvm.feature.graalvm.native)
}
----

=== Features

* GraalVM Native Image compilation (`nativeCompile` task)
* Native test execution (`nativeTest` task)
* Configurable verbose, quickBuild, fallback, and richOutput options
* Supports configuration via extension DSL or gradle.properties

=== Extension Configuration

[source,kotlin]
----
graalvmNativeConfig {
    verbose.set(false)        // Enable verbose native image output
    quickBuild.set(false)     // Enable quick build mode (faster, less optimized)
    fallback.set(false)       // Allow fallback to JVM when native image fails
    richOutput.set(true)      // Enable rich output during compilation
}
----

=== Properties Configuration

[source,properties]
----
graalvmNative.verbose=false
graalvmNative.quickBuild=false
graalvmNative.fallback=false
graalvmNative.richOutput=true
----

== Properties Reference

[cols="2,1,3"]
|===
|Property |Default |Description

|`graalvmNative.verbose`
|`false`
|Enable verbose GraalVM Native Image output

|`graalvmNative.quickBuild`
|`false`
|Enable quick build mode (faster but less optimized)

|`graalvmNative.fallback`
|`false`
|Allow fallback to JVM when native image cannot be built

|`graalvmNative.richOutput`
|`true`
|Enable rich output during native image compilation
|===

== Running Common Tasks

[source,bash]
----
# Build native executable
./gradlew nativeCompile

# Run native tests
./gradlew nativeTest
----

== Prerequisites

* GraalVM JDK (or GraalVM Community Edition)
* `native-image` tool installed (`gu install native-image` or bundled with GraalVM 22.3+)
